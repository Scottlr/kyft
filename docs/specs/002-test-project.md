# 002 - Test Project

Add the first test project and wire it into the solution.

## Scope

- Add `tests/Kyft.Tests` targeting .NET 10.
- Reference the `Kyft` project.
- Add the chosen test framework and a single placeholder test.
- Configure test files consistently with the library project.

## Acceptance

- `dotnet test` succeeds.
- The placeholder test proves the test runner is correctly configured.

## Out Of Scope

- Behavior tests for Kyft concepts.
- Test data builders or fixtures.

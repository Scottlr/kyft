Concepts

Ask deeper interval questions.

Once windows are aligned, Spanfold can measure agreement, divergence, timing, containment, coverage, point-in-time matches, and live finality without hand-written interval joins.

Compose comparator rows.

Start with overlap and residual, then add the rows needed for the incident, audit, or quality report. Each row keeps record ids and temporal ranges so the result can be exported and inspected later. 

var result = pipeline.History
    .Compare("Provider QA")
    .Target("provider-a", selector => selector.Source("provider-a"))
    .Against("provider-b", selector => selector.Source("provider-b"))
    .Within(scope => scope.Window("DeviceOffline"))
    .Using(comparators => comparators
        .Overlap()
        .Residual()
        .Missing()
        .Coverage()
        .Gap()
        .Containment()
        .Declaration("lead-lag:Start:ProcessingPosition:5"))
    .Run();

Analytics

Choose the result family that matches the decision.

Coverage and gaps

Measure how much target activity was covered and where expected coverage disappeared.

Containment

Classify whether one side fully contains, partly contains, or misses the other side's active windows.

Lead and lag

Compare start or end transitions and summarize whether the target led or lagged the comparison side.

As-of

Evaluate what matched at a specific decision point without leaking future evidence into the answer.

Live finality

Clip open windows to a horizon and keep provisional rows separate from final rows.

Concepts

Compare many sources without losing direction.

A source matrix runs directional pairwise comparisons across a provider set. Use it when the same question needs to be read across every provider pair without collapsing direction.

Source matrix

Each matrix cell has a target source and a comparison source. The row source is the target; the column source is the comparison side. That direction matters for residual, missing, lead/lag, and coverage rows. 

var matrix = pipeline.History.CompareSources(
    "Provider matrix",
    "DeviceOffline",
    ["provider-a", "provider-b", "provider-c"]);

Reading cells

Direction is part of the result.

A residual in row A, column B means A had target-only duration against B. The opposite cell asks the reverse question and can produce a different answer.

Use cases

Where temporal window analytics fits.

Spanfold is useful when teams need to explain state over time across keys, lanes, sources, phases, or pipeline stages. It is not a reporting schema; it gives applications a reliable temporal evidence layer.

Monitoring and incident management

Which provider saw the outage, and when?

Track alert, unhealthy, degraded, or missing-signal windows from multiple providers. Compare overlap, provider-only residuals, missed windows, lead/lag, and open incidents at a live horizon.
